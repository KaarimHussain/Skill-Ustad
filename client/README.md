# Skillistan Frontend

This is the frontend for Skillistan, an AI-powered skill learning platform. It's built with **React**, **TypeScript**, and **Vite** for a fast and modern development experience.

## ‚ú® Features

*   **Component-Based Architecture**: Built with reusable React components.
*   **Modern Styling**: Styled with **Tailwind CSS** and **shadcn/ui** for a clean and consistent look.
*   **Rich Animations**: Uses **Framer Motion** and **GSAP** to create smooth and engaging user interactions.
*   **Type-Safe**: Written in **TypeScript** to ensure code quality and maintainability.

## ‚öôÔ∏è Getting Started

### Prerequisites

*   [Node.js](https://nodejs.org/) (v18+)
*   An instance of the [Skillistan server](../server) running.

### Installation

1.  **Navigate to the `client` directory:**

    ```bash
    cd client
    ```

2.  **Install dependencies:**

    ```bash
    npm install
    ```

3.  **Run the development server:**

    ```bash
    npm run dev
    ```

The application will be available at `http://localhost:3000`.
